# Lessons

## 2026-03-07

- When the user corrects the media source, switch immediately to the confirmed local asset folder and stop any re-download assumption.
- For downloaded Instagram assets, inspect the filename schema before designing sort logic; these exports already encode chronology in the basename.
- When converting a creator workflow into an agent skill, treat the creator's SaaS automation tool as source evidence, not as a mandatory implementation choice; prefer deterministic `Playwright MCP + Python CLI` execution when the user wants coding-agent-friendly automation.
- For blocked agent pipelines, expose the decision contract explicitly: state the taxonomy, present exactly two recommended options, and preserve a third custom-direction branch instead of inventing hidden fallback behavior.

## 2026-03-31

- Before attempting a repo create or push for a standalone project folder, confirm the actual GitHub owner/repo from the user instead of trusting placeholder metadata like `twc-vault/glam-cli`.
